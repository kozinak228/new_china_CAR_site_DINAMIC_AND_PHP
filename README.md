# ChinaCars - Каталог автомобилей из Китая

Этот проект представляет собой динамический веб-сайт для продажи автомобилей из Китая.
Сайт позволяет просматривать каталог автомобилей, фильтровать их по параметрам, просматривать подробную информацию о каждой машине, а также управлять контентом через административную панель.

## Функциональность

### Публичная часть
- **Каталог автомобилей:** Просмотр списка доступных автомобилей с пагинацией.
- **Фильтрация:** Поиск по бренду, типу кузова и цене.
- **Детальная страница:** Полная информация об автомобиле, характеристики, свайп-слайдер фотографий (Swiper.js).
- **Комментарии:** Возможность оставлять отзывы c уникальными лайками (требуется авторизация).
- **Живой поиск:** Поиск по автомобилям без перезагрузки страницы (AJAX).
- **Сравнение:** Наглядное сравнение выбранных автомобилей.
- **Избранное:** Сохранение машин в личный кабинет.

### Административная панель
- **Управление автомобилями:** Добавление, редактирование, удаление, публикация/скрытие фотографий.
- **Управление брендами:** Создание и редактирование брендов.
- **Управление пользователями:** Просмотр и редактирование пользователей.
- **Модерация комментариев:** Управление отзывами с помощью модальных окон и удобной пагинации.
- **Интерфейс:** Встроена красивая анимированная тема "Матрица" для администраторов.

## Технологии
- **Backend:** PHP 8+ (без фреймворков, нативный код pattern MVC-like).
- **Database:** MySQL.
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5, Tailwind CSS, Swiper.js.
- **Стилизация:** Кастомный premium-дизайн с поддержкой светлой/тёмной темы по всему сайту.

## Установка и запуск
1. Клонируйте репозиторий.
2. Импортируйте базу данных из файла `dinamic-site.sql` (если есть) или создайте структуру вручную.
3. Настройте подключение к БД в файле `app/database/connect.php`.
4. Запустите локальный сервер (например, через XAMPP или `php -S localhost:8000`).
